# Requirements: MedTriads v2.1

**Defined:** 2026-01-20
**Core Value:** Users can quickly test and reinforce their knowledge of medical triads through satisfying, game-like quiz sessions

## v2.1 Requirements

### Design System

- [x] **DS-01**: Token file with colors, typography, spacing, radius, shadows, motion values
- [x] **DS-02**: Theme layer mapping semantic tokens (surface, text, brand, success, warning, danger)
- [x] **DS-03**: Icon wrapper component using Solar Icons with standardized sizes (16, 20, 24)
- [x] **DS-04**: Text primitive component using typography tokens
- [x] **DS-05**: Surface primitive component using color and shadow tokens
- [x] **DS-06**: Button component with all states (default, pressed, disabled, loading)
- [x] **DS-07**: Badge component for status indicators
- [x] **DS-08**: Tag component for labels and categories
- [x] **DS-09**: Card component with consistent styling
- [x] **DS-10**: All existing screens updated to use design tokens only
- [x] **DS-11**: All existing icons replaced with Solar Icons via wrapper
- [x] **DS-12**: DESIGN_SYSTEM.md documentation with tokens, rules, examples

### Study Mode

- [x] **SM-01**: Study Mode entry point visible on Home screen
- [x] **SM-02**: Untimed quiz flow with no countdown timer
- [x] **SM-03**: Calm visual tone distinguishing from timed mode
- [x] **SM-04**: Immediate feedback with explanation shown after each answer
- [x] **SM-05**: Progress indicator showing questions completed
- [x] **SM-06**: "Mark as tricky" button to flag questions for review
- [x] **SM-07**: Per-topic session summary stored locally
- [x] **SM-08**: Tricky questions list accessible from Library or Progress

### Challenge

- [x] **CH-01**: "Challenge a Friend" button on Results screen
- [x] **CH-02**: Styled share card showing score with competitive messaging
- [x] **CH-03**: System share sheet integration for sharing

### Category Mastery

- [x] **CM-01**: Track correct/incorrect answers per medical category
- [x] **CM-02**: Category mastery data persisted locally
- [x] **CM-03**: Category mastery cards displayed on Home screen
- [x] **CM-04**: Visual progress indicator per category (progress bar or percentage)

### App Store

- [x] **AS-01**: App icon at all required sizes including 1024x1024
- [x] **AS-02**: Screenshots for required iPhone device sizes
- [x] **AS-03**: Privacy policy URL prepared
- [x] **AS-04**: App Privacy questionnaire answers documented
- [x] **AS-05**: App Store description and keywords finalized
- [x] **AS-06**: Category and age rating determined
- [x] **AS-07**: Bundle ID and versioning finalized
- [x] **AS-08**: Release checklist (no placeholders, no debug menus, crash-free)
- [x] **AS-09**: PRIVACY_SUMMARY.md documenting data practices
- [x] **AS-10**: SCREENSHOT_PLAN.md listing screens and device sizes

## Out of Scope

| Feature | Reason |
|---------|--------|
| Deep link challenges | App not live on App Store yet |
| Challenge tracking/history | Simple share card sufficient for v2.1 |
| User accounts | No backend |
| Cloud sync | No backend |
| Leaderboards | No backend |
| Reverse quiz mode | Defer to v3.0 |
| Daily challenges | Defer to v3.0 |
| iPad/Mac support | iPhone only |
| Android | iOS first |
